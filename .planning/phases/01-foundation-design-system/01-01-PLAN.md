<task id="01-01-T1">
<title>Scaffold puma-frontend with Vite + React + TypeScript</title>
<action>
Run from the project root (`c:\Users\Hashvanth\Puma-X`):

```powershell
npm create vite@latest puma-frontend -- --template react-ts
cd puma-frontend
npm install
npm install framer-motion@11 react-router-dom@6 zustand@4 lucide-react
npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init -p
npm install @react-three/fiber @react-three/drei three
npm install -D @types/three
```

After installation, configure `vite.config.ts` to add path alias `@` → `./src`:

```ts
// vite.config.ts
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import path from 'path';

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
  server: {
    port: 5173,
    proxy: {
      '/api': 'http://localhost:3001',
    },
  },
});
```

Add to `tsconfig.json` under `compilerOptions`:
```json
{
  "paths": { "@/*": ["./src/*"] },
  "strict": true
}
```
</action>

- `puma-frontend/tailwind.config.js` exists at root of puma-frontend/

<action>

```powershell
New-Item -ItemType Directory -Name puma-backend
cd puma-backend
npm init -y
npm install express cors dotenv @supabase/supabase-js
npm install -D typescript @types/express @types/node @types/cors ts-node nodemon
npx tsc --init
```

Update `puma-backend/tsconfig.json`:
```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "rootDir": "./src",
    "outDir": "./dist",
    "strict": true,
    "esModuleInterop": true,
    "resolveJsonModule": true
  }
}
```

Create `puma-backend/src/db/supabase.ts`:
```ts
import { createClient } from '@supabase/supabase-js';
import dotenv from 'dotenv';
dotenv.config();

export const supabase = createClient(
  process.env.SUPABASE_URL ?? '',
  process.env.SUPABASE_ANON_KEY ?? ''
);
```

Create `puma-backend/src/routes/health.ts`:
```ts
import { Router } from 'express';
const router = Router();
router.get('/health', (_req, res) => {
  res.json({ status: 'ok', timestamp: new Date().toISOString() });
});
export { router as healthRouter };
```

Create `puma-backend/src/routes/shoes.ts` (placeholder):
```ts
import { Router } from 'express';
const router = Router();
router.get('/', (_req, res) => {
  res.json({ message: 'Shoe catalog — Phase 4' });
});
export { router as shoesRouter };
```

Create `puma-backend/src/routes/recommend.ts` (placeholder):
```ts
import { Router } from 'express';
const router = Router();
router.post('/', (_req, res) => {
  res.json({ message: 'Recommendation engine — Phase 4' });
});
export { router as recommendRouter };
```

Create `puma-backend/src/index.ts`:
```ts
import express from 'express';
import cors from 'cors';
import dotenv from 'dotenv';
import { healthRouter } from './routes/health';
import { shoesRouter } from './routes/shoes';
import { recommendRouter } from './routes/recommend';

dotenv.config();
const app = express();
const PORT = process.env.PORT ?? 3001;

app.use(cors({ origin: process.env.FRONTEND_URL ?? 'http://localhost:5173' }));
app.use(express.json());

app.use('/api', healthRouter);
app.use('/api/shoes', shoesRouter);
app.use('/api/recommend', recommendRouter);

app.listen(PORT, () => {
  console.log(`PUMA Backend running on http://localhost:${PORT}`);
});
```

Update `puma-backend/package.json` scripts:
```json
{
  "scripts": {
    "dev": "nodemon --exec ts-node src/index.ts",
    "build": "tsc",
    "start": "node dist/index.js"
  }
}
```

Create `.env.example` at project root and in puma-backend/:
```
SUPABASE_URL=your_supabase_url_here
SUPABASE_ANON_KEY=your_supabase_anon_key_here
PORT=3001
FRONTEND_URL=http://localhost:5173
```

Create `.gitignore` at project root:
```
node_modules/
.env
dist/
.DS_Store
*.local
```
